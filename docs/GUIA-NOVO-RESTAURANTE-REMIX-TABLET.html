Guia completo — Novo restaurante (Remix) + Tablet

Euro Business Group · SnapOrder / Kebab Turco Master
Versão: Junho 2026 · Para clonar um restaurante novo na Lovable e ligar o tablet da loja.

O que vai conseguir no fim

  1. Um restaurante novo online (site + painel), com nome, logo e cores do cliente.
  2. Um ficheiro de instalação (APK) no tablet — a app da cozinha/mostrador.
  3. RustDesk instalado — para aceder ao tablet à distância.
  4. Não precisa de copiar a pasta do projeto para o tablet.

PARTE 1 — Remix na Lovable

1.1 Fazer o Remix

  1. Abre lovable.dev e entra no projeto Kebab Turco SnapOrder (Master).
  2. Menu do projeto → Remix this project.
  3. Escolhe um nome (ex.: Pastelanche SnapOrder).
  4. Confirma — nasce um projeto novo e separado.
O Remix copia o aspeto e o código. Não copia pedidos, clientes, base de dados cheia nem chaves Stripe.

1.2 Ativar a nuvem

  1. No projeto novo: More → Cloud.
  2. Ativa a nuvem — cria a base de dados desse restaurante.

1.3 Script de arranque (obrigatório)

  1. More → Cloud → SQL Editor.
  2. Copia todo o ficheiro BOOTSTRAP_MASTER_TEMPLATE.sql do projeto Master.
  3. Cola no editor e clica Run.

1.4 Personalizar

ItemOnde
NomeAdmin → Configurações
Logo e coresAdmin → Branding
CardápioAdmin → Cardápio
HoráriosAdmin → Horários
EntregaAdmin → Entrega

1.5 Segredos e Publish

Em Cloud → Secrets: Stripe, webhooks, IA, notificações. Depois clica Publish e anota o endereço (ex.: https://pastelanche.net).

PARTE 2 — Gerar o APK (app do tablet)

Importante: O tablet usa o APK, não a pasta do projeto no Mac.

2.1 URL do restaurante novo

No ficheiro capacitor.config.ts, altera:

server: {
  url: "https://SEU-DOMINIO-AQUI.net",
}

2.2 Compilar

npm install
npm run build
npx cap sync android
npx cap open android

No Android Studio: Build → Build APK(s). O ficheiro fica em android/app/build/outputs/apk/release/.

Renomeia para algo claro, ex.: Pastelanche-SnapOrder-tablet-v1.apk

PARTE 3 — RustDesk (acesso remoto)

RustDesk permite ver e controlar o tablet à distância — instalar APK, configurar impressora, resolver problemas.

Não copies a pasta inteira do projeto para o tablet. Isso não instala a app.

3.1 No Mac

  1. Site: https://rustdesk.com
  2. Instala versão macOS.
  3. Anota o teu ID e define palavra-passe.

3.2 No tablet Android

  1. Instala RustDesk (Play Store ou site oficial).
  2. Anota o ID do tablet e define palavra-passe.
  3. No Mac: liga ao ID do tablet.

3.3 Enviar o APK

  1. RustDesk → Transferência de ficheiros.
  2. Envia só o ficheiro .apk (~30–80 MB).
  3. No tablet: pasta Download → toca no APK → Instalar.

PARTE 4 — Configurar o tablet

  • Wi‑Fi estável e tablet ligado à corrente.
  • Abre a app — deve mostrar o site do restaurante novo.
  • Configura impressora no Admin e faz pedido de teste.

Checklist final

  • Site abre no telemóvel
  • Cliente faz pedido (levar / mesa / entrega)
  • Painel e cozinha recebem o pedido
  • APK instalado com o site certo
  • Impressão funciona
  • RustDesk instalado (ID guardado)
  • Nome, logo, cores e preços corretos

Erros comuns
